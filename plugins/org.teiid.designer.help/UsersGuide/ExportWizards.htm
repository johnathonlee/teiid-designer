Export Wizards

Teiid Designer contributes the following 2 export wizards:

  • Data Definition Language (DDL) file
  • Model Project Set

DDL Exporter

The DDL Export wizard provides a means to create a DDL file based on the structural contents of a relational model in your workspace.

To export DDL:

Step 1: Open Export    wizard, select the Data Definition Language (DDL) file export option and click Next>.

Step 2: On the first wizard page select structural components to export, set desired options and click Finish.
