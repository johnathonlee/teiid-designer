Editing Model Objects

This section summarizes Designer's features for editing existing model objects contained in your models.

The primary actions for editing model objects are:

  • Cut - Deletes the selected object(s) and copies it to the clipboard.
  • Copy - Copies the selected object(s) to the clipboard.
  • Paste - Pastes the contents of the clipboard to the selected context.
  • Clone - Duplicates the selected object in the same location with the same name; user is able to rename the new object right in the tree.
  • Delete - Deletes the selected object(s).
  • Rename - Allows a user to rename an object.

These actions are presented in Designer's main Edit menu and also in the right-click context menus for model objects selected in the Model Explorer, Diagram Editor and Table Editor.

Editing Special Model Objects

Designer provides specialized object editors to handling a few complex model objects and their unique properties. These objects include:

  • Transformation
  • Mapping Class Input Set
  • XML schema choice element
  • Recursive Mapping Class

The details for editing these objects is in the following sections.

Edit Transformation

To edit a transformation you can:

  1. Double-click Edit
    • A relational view table or procedure in the Model Explorer or Diagram Editor
    • A transformation node in a transformation diagram or mapping transformation diagram
    • A mapping class in a mapping diagram or mapping transformation diagram
  2. Right-click Edit action on selected object in the Model Explorer, Diagram Editor or Table Editor
    • A relational view table or procedure
    • A transformation node in a transformation diagram or mapping transformation diagram
    • A mapping class in a mapping diagram or mapping transformation diagram

If a Model Editor is not currently open for the selected object's model, a Model Editor will be opened.

After the corresponding transformation diagram is opened in the Diagram Editor, the Transformation Editor is displayed in the lower section of the Diagram Editor.

Edit Input Set

To edit an input set you can:

  1. Double-click Edit an input set in a mapping transformation diagram
  2. Right-click Edit an input set selected in a mapping transformation diagram
  3. Click the button on the input set in the mapping transformation diagram

The result of the Edit action is to open the Input Set Editor in the lower section of the Diagram Editor.
