Designer Example Projects

Teiid Designer provides project examples through JBoss's Help > Project Examples... action framework.

After launching the wizard, open the Teiid Designer category and select your desired example project.

This action will load a copy of these examples into your workspace in order to familiarize yourself with the content and structure of projects and models.
