Creating a Relational Model from WSDL

You can create a Relational model by selecting an existing Web Services Connection Profile defined by a WSDL file in your workspace or defined by a URL. Designer will interpret the WSDL, locate any associated or dependent XML schema files, generate an XML View of the schema components and create a Web service model representing the interfaces and operations defined in the WSDL.

To create a Web Service model from a WSDL file located in your workspace:

Step 1: Choose the File > Import action in the toolbar or select a project, folder or model in the tree and choose "Import..."

Step 2: Select the WSDL into Relational Model option shown below and click NEXT >.

Step 3: Select an existing Web Services Connection Profile (or create New...)  specificy workspace location for your new model and click OK.


Step 4: The next page "WSDL Operations Selection" allows customizing the resulting content of your Web Service model by selecting/deselecting various operations and interfaces in the following dialog.


Step 5: Select "Finish" to complete.
