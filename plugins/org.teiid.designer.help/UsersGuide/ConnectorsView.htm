Teiid View

The Teiid View provides a means to display and manage Teiid server instances within Designer.

To show the Teiid View click "Views > Show View > Other..." to display the Show View dialog. Choose "Teiid Designer > Teiid " view and hit OK.

NOTE: Documentation for this feature is under development
