Teiid Designer Users Guide

Teiid Designer is an Eclipse-based graphical modeling tool for modeling, analyzing, integrating and testing multiple data sources to produce Relational, XML and Web Service Views that expose your business data.

This document provides detailed and summary information about the Teiid Designer including:

  • Detailed descriptions and screen caps of views, dialogs, menus and actions
  • Step-by-step wizard instructions for complex operations
  • Background summaries of key concepts

For more information visit:

  • Teiid and Teiid Designer - http://www.teiid.org
  • JBoss - http://www.jboss.org
  • Red Hat - http://www.redhat.com
