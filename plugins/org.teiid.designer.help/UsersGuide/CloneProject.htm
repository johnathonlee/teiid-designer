Clone Project

Teiid Designer provides the ability to create a designer model project that is structurally identical to an existing model project including directories as well as XMI and XSD model files.

Notes:

  • This action clones a complete model project containing any number of model (XMI or XSD) files organized in a user-defined directory structure.
  • All object references (UUID's) within the original project will be replaced with new unique references.
  • Any model dependencies or internal object references are refactored to reflect the dependencies within the cloned project.
  • Any model references to models in projects external to the original project will NOT be replaced.
  • Only XMI and XSD files are cloned. All other file types in your project will NOT be processed nor copied into your newly cloned project.
  • If one or more editors that require "save" are open, the user will be asked to save them before continuing with the cloning process.

How do I clone a project?

To clone a model project:

Step 1: Select an existing model project in the Model Explorer.

Step 2: Right-click select the Clone Project action or select the Project > Clone Project action located in the Designer’s main menu bar.

Step 3: On the Clone Project wizard page, provide a name for your new project.

Step 4: (Optional) If you wish to create your cloned project in location other than your default workspace location, uncheck the ï¿½Use defaultï¿½ checkbox and specify (type in or browse to) a new directory location on your local file system.

Step 5: Click Finish to generate your new project.
