Creating a Virtual Database

Teiid Designer allows you to create and edit VDB's. (see Virtual Database Summary ). To create a new VDB:

Step 1: Select the new Teiid VDB action by:

  • Choosing the File > New... > Teiid VDB action
  • Selecting a project or folder in the Model Explorer and choosing the same action in the right-click menu.
  • Selecting the New button on the main toolbar and selecting the Teiid VDB action
  • Typing the keystrokes SHIFT+ALT+N and selecting the Teiid VDB action

Step 2: In the New VDB dialog’s In Folder: field specify the directory on your file system in which the new VDB will be located.  Click OK. 

Step 3: The new empty VDB will be created and opened in a VDB Editor.

In addition to creating a new VDB, you can test it within Designer. See Execute VDB.

By selecting one or more models in your Model Explorer view and launching the New VDB dialog, the dialog now shows the select model(s) and automatically adds them to the new VDB.
