Import Metadata from Text Files

The Designer provides various import options for translating specific text file metadata into models. This is accomplished via the Import > Metadata from Text Files on file system option.

To import from text files:

Step 1: In Designer , choose the File > Import action in the toolbar or select a project, folder or model in the tree and choose "Import..."

Step 2: Select the import option Metadata from Text Files on file system and choose NEXT>

Step 3: Select an import type via the drop-down menu shown below.

These steps required for each type are defined below:

  • Relational Tables
  • Relationships
  • Relational Virtual Tables
  • Datatypes

Importing Relational Table

To create relational tables from imported text metadata, perform Steps 1 thru 3 (above) and select the Relational Tables import type, then click Next >

Step 4: In the next page, you'll need to provide a source text file containing the metadata formatted to the specifications on the previous page.

Step 5: Select the target location in which the new relational entities will be created. The default datatype for created columns may be set via the options checkbox

Step 6: Click Finish to perform the import.
