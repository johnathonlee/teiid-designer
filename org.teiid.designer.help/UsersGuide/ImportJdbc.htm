Importing Sources via JDBC

Source relational models can be created by importing from JDBC sources.

Follow the steps below...

Note: Depending the detail provided in the database connection url information and schema, Steps 5 thru 7 may not be required

Step 1: In Designer , choose the File > Import action in the toolbar or select a project, folder or model in the tree and choose "Import..."

Step 2: Select the import option Metadata from JDBC Database and choose NEXT>

Step 3: Select existing or previous Source configuration or press "Sources..." to create new source configurations.

Step 4: After selecting Source, input password and press "Connect to Database". Press NEXT> (or FINISH> if enabled)

Step 5: On the "Select Database Metadata" page, choose select the types of objects in the database to import. Press NEXT> when finished (or FINISH> if enabled)

Step 6: On the "Select Database Objects" page, view the contents of the schema, or change selections. Select which database schema objects will be used to construct relational objects. Press NEXT> when finished (or FINISH> if enabled)

Step 7: On the "Specify Import Options" page, specify desired Model Name as well as any other options used to customize the constructed relational objects. Press FINISH> to complete.

 

During the "Finish" processing, a monitor will be displayed providing feedback on the import progress.
