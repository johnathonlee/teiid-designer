Create Relational View Model from XML Schema

In addition to creating relational source models from XML Schema files on your file system, you can also create relational view models from XML Schema files in your VDB. The resulting relational tables will match the basic data structure of your XML Schema.

Step 1: In the Model Explorer, select a desired XML Schema model.

Step 2: Right-click and choose the Models > Create Relational View Model from Schema action.

Step 3: Specify an existing location in your workspace for your new model via the Browse... button.

Step 4: Provide a valid model name unique within your workspace.

Step 5: Select one or more XML Schema global entities and move them to the right panel. Move individually using the > button, or move all entities via the >>> button.

Step 6: Click Finish.  After generation your new model can be found in your location specified in Step 3.
