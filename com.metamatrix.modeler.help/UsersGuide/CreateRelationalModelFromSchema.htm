Create Relational Source Model from XML Schema

In addition to creating relational source models by importing your metadata (Import Relational Data ) you can also turn your existing XML Schema into relational source or view models. You can create XML View models using the steps below...

Step 1: Choose the File > Import action in the toolbar or select a project, folder or model in the tree and choose "Import..." action to launch the Import Wizard.

Step 2: Select option XML Schema as Relational Source then Click NEXT>.

Step 3: On the next page, specify all options for defining document source and catalog options then Click NEXT>.

Step 4: On the XML Schema File Selection page, choose a schema file located in your Workspace... or a file on your local File System . Click Next >.

Step 5: Select elements that will be used as root elements in your XML documents.

Step 6: Click Finish.  After generation the new model can be found in your VDB’s View folder.
