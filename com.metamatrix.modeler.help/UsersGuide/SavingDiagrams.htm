Saving Diagrams

The Designer allows you to save the current diagram as an image file in either jpg or bmp format.

To save a diagram to file, either right-click on diagram and choose Save Diagram To File action or choose the associated diagram toolbar button . This opens the Save Diagram Image as File dialog shown below.

In this dialog you can specify a folder, file name and file type. The jpg and bmp file types are supported.

If no objects in the diagram are selected, the entire diagram (not just the visible part) will be in the saved image. 

If any of the objects in the diagram are selected, the saved image will be as large as smallest rectangle of the diagram that contains the selected objects.
