Editing Connector Bindings

A valid Virtual Database (VDB) contains connection information necessary to query the server. This connection information is managed via the Connector Bindings Editor found on the Execute tab in the VDB Editor

Each JDBC Source model requires an associated valid connector binding. Select the edit button to change or edit bindings. Properties for a selected binding can also be changed individually in the Connector Bindings Properties table.

To assign a connector binding to a source model:

Step 1: Open VDB Editor and select the Execute tab

Step 2: Select a row in the Connector Bindings Editor table, and choose the button on the right.

Step 3: A selection dialog appears containing available connector bindings. Select desired binding and press "OK". New bindings can be created by using the New Binding or Import Source (if available) tabs of the dialog.

Note: Modifying connector bindings will render the editor as "changed", so a Save operation is required to permanently save these changes.
