You can open the Configuration Manager by:

  • Selecting a model in the Model Explorer and right-clicking the Models... > Open Configuration Manager action
  • Select the Open Configuration Manager action on the main toolbar
  • Open an existing VDB and select the link Open the configuration manager... in the Execution Status section .

Editor Components

The Configuration Manager editor contains two primary sections, Configuration and Extensions as shown below.

Managing Connectors

You can manager your connectors by importing a server configuration or connector bindings. You can use the import links at the top of the Connectors section or use the select-right-click menus to manager your connectors.

Managing Connector Extensions

You can extend your connector functionality by importing additional connector bindings. You can use the import links at the top of the Connection Modules section or use the select-right-click menus to manager your connector extensions.

Editing User Defined Functions

You can edit your user-defined functions via the Configuration Manager. Just select the link Open the function model in the UDF Extension Modules section.

The Function Model should then be opened in an editor displaying the model's package diagram.
