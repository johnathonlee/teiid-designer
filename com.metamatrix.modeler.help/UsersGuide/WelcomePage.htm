Designer's Welcome Page

When you start Designer for the first time, a Welcome page will be displayed. The page contains several links or actions to get you started on common tasks in Designer. A description for each action is provided on the Welcome page. Note that on the Linux version of Designer, the Welcome page will look a little different.

 

Import

  • Import Metadata
  • Open Existing Model Projects
  • Open the MetaMatrix Repository Perspective
  • Import Designer Examples

Create

  • Create a Model Project
  • Create a Metadata Model
  • Create a Virtual Database Definition

Learn

  • Designer's User Guide
  • Learn more about the Eclipse Workbench

You can close this view at any time and open it again using the Help > Welcome action.

As you begin modeling in your data, the Welcome page will initially become docked to the right side of your workbench. As with any Eclipse view, you can drag and re-dock it in a different postion.
