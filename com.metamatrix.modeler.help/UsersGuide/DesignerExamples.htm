Designer Example Projects

The Designer comes with existing projects containing example models and VDBs. You can load a copy of these examples into your workspace to familiarize yourself with the content and structure of projects and models.

You can import these examples by opening the Help > Welcome page and selecting the Samples action shown below.

To import an example project set:

Step 1: Select the Federate Designer Models action on the Samples page to display the Import Model Project Set dialog.

Step 2: Select one of the available examples (i.e. books or parts)

Step 3: Specify a target folder to copy this project into. (usually your installation's workspace folder). Type in the location or select Browse.. to select a folder on your local file system.

Step 4: Change/set any desired options

Step 5: Select OK to complete the import.
