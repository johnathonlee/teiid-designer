Creating a Virtual Database

Federate Designer allows you to create and edit VDB's. (see Virtual Database Summary ). To create a new VDB:

Step 1: Select the New Virtual Database action by:

  • Choosing the File > New... > Virtual Database Definition action
  • Selecting a project or folder in the Model Explorer and choosing the same action in the right-click menu.
  • Selecting the New button on the main toolbar and selecting the Metadata Model action

Step 2: In the New VDB dialog’s Location: field specify the directory on your file system in which the new VDB will be located.  Click OK. 

Step 3: The new empty VDB will be created and opened in a VDB Editor.

In addition to creating a new VDB, you can test it within Designer. See Execute VDB.
