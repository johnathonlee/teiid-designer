Diagram Link Router Options

Relationships (PK-FK) displayed in package or custom diagrams can be customized via a router style preference. When a diagram is first displayed, it utilizes the current/default router style preference (Orthogonal, Direct or Manual) (see Designer Preferences).

Once displayed, the router option can be changed for that particular diagram via the right-click "Link Types" action (See below). Each diagram can have it's own router style defined. Examples of each type are shown below.
