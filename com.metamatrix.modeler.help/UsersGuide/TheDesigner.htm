The Designer

Federate Designer is an Eclipse-based graphical modeling tool for modeling, analyzing, integrating and testing multiple data sources to produce Relational, XML and Web Service Views that expose your business data.

Designer is comprised of a set customized Eclipse-compatable features including:

  • perspectives
  • views
  • editors
  • dialogs
  • wizards

These components are intended to provide a rich intuitive modeling experience.
